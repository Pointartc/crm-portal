# NexusCRM — Premium Dark SaaS CRM Portal ⚡

A high-end, responsive enterprise CRM platform built with the latest **Angular 18+**, featuring **Tailwind CSS** for a luxury dark-themed design and fully powered by **Angular Signals** for lightning-fast dynamic data reactivity.

---

## ✨ Key Features

- 🔒 **Secure JWT Authentication Module** — Form validation, dynamic loading spin states, and automatic error handling connected to DummyJSON API.
- 🧱 **Monolithic Layout Shell** — Premium desktop and tablet responsive design with a glassmorphism Sidebar navigation and global Header context.
- 📊 **Dynamic KPI Dashboard** — Real-time reactive counters displaying total leads, task completion metrics with auto-filling progress bars, and custom monthly performance charts.
- 👥 **Interactive Leads Management** — A gorgeous dark SaaS data table featuring an optimized search engine utilizing Angular `computed()` signals for zero-latency keystroke filtering.
- ✅ **Operational Tasks & Todo List** — Fully interactive checklist with state persistence toggles, optimistic UI updates, and colorful priority status pills (`In Progress` / `Completed`).
- 🛡️ **Advanced Router Protection** — Complete guard (`canActivate`) architecture to prevent unauthorized route manipulation, integrated with an automatic token-carrying `authInterceptor`.

---

## 🛠️ Tech Stack & Architecture

- **Framework:** Angular 18+ (Strict Standalone Components Architecture)
- **State Management:** Reactive Angular Signals & Computed State Pipelines
- **Styling:** Tailwind CSS (Glassmorphism layout, modern gradient accents, active focus glows, responsive utility classes)
- **Data Source:** DummyJSON API Integration via functional `HttpClient` streams
- **Version Control:** Git Workflow structured for corporate standards

## 🏎️ Getting Started

Follow these quick steps to launch the repository locally on your computer.

### 1. Clone the repository
```bash
git https://github.com/Pointartc/crm-portal.git
cd crm-portal
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the application
```bash
ng serve
```
Open your browser and navigate to `http://localhost:4200/`.
